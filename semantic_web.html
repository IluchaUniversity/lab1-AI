Семантичний Веб

Короткий опис

Семантичний Веб - це розширення традиційного Вебу, яке дозволяє машинах розуміти та реагувати на даних, які знаходяться в Інтернеті, шляхом використання стандартів та маркерів, що додають семантичний зміст до інформації.

Стандарти, методи, програмне забезпечення, сфери застосування

Стандарти:

  • RDF (Resource Description Framework)
  • OWL (Web Ontology Language)
  • SPARQL - мова запитів для RDF

Програмне забезпечення:

  • Protégé
  • Jena Framework

Сфери застосування:

  • Знаннєві бази
  • Пошукові системи нового покоління
  • Інтелектуальний аналіз даних
