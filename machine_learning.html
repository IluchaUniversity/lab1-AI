Машинне навчання

Короткий опис

Машинне навчання - це підгалузь штучного інтелекту, що фокусується на розробці алгоритмів, які дозволяють комп'ютерам вчитися виконувати завдання без специфічного програмування для цих завдань.

Типи, алгоритми, програмне забезпечення, сфери застосування

Типи:

  • Наглядоване навчання
  • Ненаглядоване навчання
  • Навчання з підкріпленням

Алгоритми:

  • Лінійна регресія
  • Дерева рішень
  • Нейронні мережі
  • Кластеризація K-means

Програмне забезпечення:

  • Scikit-learn
  • TensorFlow
  • PyTorch

Сфери застосування:

  • Обробка зображень
  • Прогнозування даних
  • Рекомендаційні системи
